Z-Metronome is a simple app for Apple Watch, iPhone and iPad that provides a pulse indication between 40 and 200 beats per minute (BPM) using 3 independent ways of being indicated: 1) Sound, 2) Sight, 3) Feel .

Promotional Text:

Need a metronome that only you can see and hear and feel from your wrist or next to you? Useful if you are performing, conducting, or practicing.
